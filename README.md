# FR_effort
Учебный проект. Пайплан распознавания лиц. 
Проект представляет собой попытку реализации пайплайна распознавания лиц. 

Цель проекта: 
- написать и обучить нейросеть определять лица на изображении, выравнивать его по трем ключевым точкам (левый и правый глаз, нос), и определять сходство поданного изображения с базой данных (идентифицировать лицо на изображении). 

Включает в себя: 
1. анализ аттрибутов датасета CelebA и формирование датасета для обучения модели п. 2 (Creaion_of_dataset_local.ipynb)
2. обучение модели подготавливающей фотографии к этапу 3: обучению модели распознавания лиц. Этап 2 включает: обрезка по bounding box, предсказание ключевых точек лица, выравнивание изображения по 3 точкам из 5 - так выходит стабильнее, и изображение меньше искажается (Hourglass_model_notebook.ipynb).
3. Импорт модели ResNet 18 (как наиболее доступной по мощности) и тренировка ее на определение класса (принадлежность изображения к какому-либо ID датасета)
